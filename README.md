# matrix
C++ Matrix template

- Bareiss method matrix determinant calculation 
- Random matrix with known determinant value creation
- Cmake project with gtest framework support

### HOW TO BIULD
mkdir build
cd build
cmake ..
cmake --build .

### HOW TO RUN
From build directory :
./matrix_test

### HOW TO RUN UTIT TESTS
From build directory :
./tests/unit_tests
